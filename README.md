# Analisis-Fourier-Matlab
# Análisis de Señales y Transformada de Fourier utilizando MATLAB

## Descripción

Este proyecto tiene como objetivo analizar señales en el dominio del tiempo y de la frecuencia utilizando MATLAB y la Transformada Rápida de Fourier (FFT).

Se implementaron diferentes señales básicas como la señal senoidal, el pulso rectangular y la función escalón. Posteriormente se calculó su Transformada de Fourier para estudiar su comportamiento en el dominio de la frecuencia.

## Objetivos

- Representar señales en el dominio del tiempo.
- Aplicar la Transformada de Fourier mediante FFT.
- Visualizar la magnitud y fase del espectro de frecuencia.
- Analizar las propiedades de linealidad, desplazamiento temporal y escalamiento.
- Comparar la representación de las señales en ambos dominios.

## Herramientas utilizadas

- MATLAB
- FFT (Fast Fourier Transform)

## Señales analizadas

- Señal senoidal
- Pulso rectangular
- Función escalón

## Propiedades verificadas

- Linealidad
- Desplazamiento temporal
- Escalamiento en frecuencia

## Archivos del proyecto

## Resultados

Los resultados muestran cómo cada señal posee una representación distinta en el dominio de la frecuencia y permiten verificar experimentalmente las propiedades fundamentales de la Transformada de Fourier.
